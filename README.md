📌 About

YashhMaceOptimized is a client-side mod designed to improve mace combat in Minecraft 1.21.8.
This free version provides essential combat optimizations to give you a better PvP experience without using any unfair advantages.

⚠️ This is NOT a hack client.
All features are designed to remain within legit gameplay standards.
